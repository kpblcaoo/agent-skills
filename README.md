# Claude Skills

Локальный набор инструкций для AI-агента.

Здесь лежат общие правила работы и отдельные skills под типовые сценарии: планирование, разбиение задач, короткие ответы, проверка идей и настройка базовых guardrails.

## `CLAUDE.md`

`CLAUDE.md` - общие правила поведения агента в этом репозитории.

Он задает стиль работы: проверять факты, не выдумывать, не расширять задачу без причины, делать минимальные безопасные изменения и сообщать, что именно было проверено.

Смысл файла - держать агента в рамках: меньше лишней архитектуры, больше конкретики, проверок и честных ограничений.

## Skills

Каждый skill лежит в отдельной папке в файле `SKILL.md`.

- `caveman` - режим экономии слов. Нужен, когда ответы должны быть максимально короткими, но технически точными.
- `grill-me` - интервью по идее, плану или дизайну. Нужен, чтобы найти слабые места до реализации.
- `make-plan` - превращает задачу или обсуждение в короткий план работ. Нужен перед нетривиальной реализацией.
- `slice-plan` - режет готовый план на маленькие проверяемые шаги. Нужен, чтобы работу можно было делать и ревьюить по частям.
- `implement-slice` - выполняет ровно один утвержденный шаг. Нужен, чтобы не расползаться за пределы выбранной задачи.
- `setup-repo-guardrails` - помогает добавить минимальные проверки качества и безопасности. Нужен, чтобы агентские изменения было проще проверять.
