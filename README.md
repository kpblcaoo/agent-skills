# Claude Skills

Репозиторий с локальными Claude Skills для повседневной работы.

Каждый skill лежит в отдельной папке и содержит файл `SKILL.md` с названием, описанием и правилами использования.

## Что здесь есть

- `caveman` - режим очень коротких ответов без лишней воды.
- `grill-me` - строгий опрос по плану или дизайну до общего понимания.
- `make-plan` - короткий план реализации по задаче или обсуждению.
- `slice-plan` - разбиение утвержденного плана на маленькие проверяемые слайсы.
- `implement-slice` - реализация одного утвержденного слайса без лишней работы.
- `setup-repo-guardrails` - предложение или настройка базовых quality/security guardrails для репозитория.

## Структура

```text
.claude/skills/
  README.md
  caveman/SKILL.md
  grill-me/SKILL.md
  implement-slice/SKILL.md
  make-plan/SKILL.md
  setup-repo-guardrails/SKILL.md
  slice-plan/SKILL.md
```

## Как использовать

1. Выберите подходящий skill по описанию.
2. Откройте соответствующий `SKILL.md`.
3. Следуйте инструкциям внутри skill-файла.

## Как добавлять новый skill

1. Создайте новую папку рядом с остальными.
2. Добавьте в нее `SKILL.md`.
3. Кратко опишите назначение skill и правила его применения.

## Примечание

Этот репозиторий рассчитан на локальное использование и хранение набора рабочих инструкций для агента.
